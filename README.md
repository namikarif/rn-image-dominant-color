# rn-image-dominant-color

Detect dominant color of image

## Installation

```sh
npm install rn-image-dominant-color
```

## Usage

```js
import { getDominantColor } from 'rn-image-dominant-color';

// ...

const result = await getDominantColor('imagePath or ImageUri');
```
